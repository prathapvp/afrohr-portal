# AfroHR Brand SERP 30-Day Tracker

Use this as a daily operating sheet (about 20 minutes/day) to improve branded Google result quality over 30 days.

## How To Use

- Mark each day as done when the task is completed.
- Add notes for what changed.
- Update KPI values once per week.
- Keep brand naming consistent everywhere: AfroHR.
